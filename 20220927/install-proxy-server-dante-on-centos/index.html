安装步骤

创建并编辑配置文件

vi /etc/sockd.conf

127.0.0.1 替换成 x.x.x.x/8 即可。

配置说明： https://www.inet.no/dante/doc/1.4.x/config/server.html

运行

测试代理

测试能否正常使用

可以使用 netstat -lntp 看启动状态
