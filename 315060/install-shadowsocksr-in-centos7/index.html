Linode CentOS7下面的安装流程

安装前准备

下载shadowsocksr

配置shadowoskcr

vi user-config.json

后台运行ssr

cd shadowsocks

后台运行

python server.py -d start

如果要停止/重启：

python server.py -d stop/restart

查看日志：

tail -f /var/log/shadowsocksr.log
