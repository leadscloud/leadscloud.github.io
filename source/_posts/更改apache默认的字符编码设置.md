从服务器上直接看txt文件有时会显示乱码，更改下网页编码即可，但这样对于用户不太友好。
后来发现服务器在返回响应时的编码没有设置。
在`.htaccess`文件中加上下面一句话即可。

```
AddDefaultCharset utf-8
```

没添加前的服务器响应：
```
Content-Type:text/plain;
```

添加后的服务器响应：
```
Content-Type:text/plain; charset=utf-8
```
可以看到，多了一个 `charset=utf-8`
