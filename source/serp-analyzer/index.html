域名所属人

显示搜索引擎结果页域名所属人的浏览器扩展， 适用于Chrome与Firefox

安装到Chrome 安装到Firefox

支持Google、Baidu、Bing、Yahoo等搜索引擎。



功能介绍，Functions

This extension shows a author name for the current website or search engine result page domain list. show the url author name in SERP listing item when you search some keywords, such as "silica sand crusher".

extension will request server to check the url have any author info. if get the url info, will show the url author name in search engine results list or in extension icon. if server have no url info, will not show any author info. No personal data will be transmitted or stored. However, extension handles user's settings only to analyze trends for how extension are used.

显示当前网站域名的所属人和搜索引擎搜索结果页域名所属人信息. 只有当前域名在服务端能查询到结果时都会在弹出窗口显示。

在搜索引擎结果页，每个列表的url信息会从服务器中获取域名相关信息，如果匹配到所有者，会在每个列表前显示当前域名的所有人信息。

弹出窗口显示了有关域的详细信息。详细信息是域名所属人及相关情况。 为保护你的隐私，扩展只发送网址信息。

Privacy Policy



它是如何工作的?

使用这个扩展后，你在浏览网页时会显示域名属于谁的，在右上角的图中会显示文字，点击图标能显示更详细的信息。

在你使用搜索引擎时，搜索结果页每条项目的URL信息也会在列表前头显示域名是属于谁的，方便你分析网站所有者。

由于chrome应用商店的严格审核原则，我们的扩展不会收集你的敏感信息，仅限于向服务器请求时发送域名信息.

此扩展适用于任何公司，这是属于我的个人项目，任何人都可免费使用，域名信息个人可以在扩展选项中添加，我在后台也维护一份域名所有者信息，欢迎大家补充域名信息，你提交的域名信息未来可能会供其他使用者使用.



如何提交你的域名信息？

在扩展的选项中，你可以添加域名及所属人信息，这些数据会保存到服务器中，并且其他人也可以使用这些数据.

如果你想批量添加域名信息，比如1000个，请联系我：sbmzhcn@gmail.com

批量导入域名，需要把域名及所属人数据放到csv文件中，格式如下： 

string,string,string,string,string,string
hostname,nickName,registrar,language,type,description
google.com,谷歌,,,,
第一行是字段类型，比如string、int等。第二行是字段名，不可更改，第三行开始才是数据，请按照这个格式发送到我的Email，我会及时更新到数据库中。
