域名所属人及IP信息

显示搜索引擎结果页域名所属人和域名IP信息及国旗图标的浏览器扩展， 适用于Chrome与Firefox

安装到Chrome 安装到Edge 安装到Firefox

Browser Add-on for Show domain owner & IP & country flag

The SERP Analyzer browser add-on can be easily installed on either Chrome or Microsoft Edge or Firefox. Shows domain author name and Server IP, Country Flag & Website Info in the location bar if matched.


功能介绍，Functions

Shows country flag based on the server location for any website.

Display domain name owner information and URL domain IP information, the pop-up box will display the domain ip info of the currently visited webpage, include: country flag, ipv4, ipv6, location, country, asn, port. The flag icon of the domain will be displayed on the toolbar.

On the SERP, adds the position of search results and domain owner info next to search results in Google, Bing, Yahoo, Baidu.

When obtaining the domain owner and ip information, a request will be sent to the server, and only the domain name will be sent during the request. No personal data will be transmitted or stored.

显示访问网址的IP信息和国旗图标，在搜索引擎结果页显示每项序号和域名所属人.

在SERP中（比如Google、百度、Bing搜索结果页）显示每项结果网站的对应所属人信息及排名序号，域名所属人信息先从本地规则中获取，没有则从服务器端获取。在弹出窗口中，会显示当前访问网址的IP信息，包含：服务端IP、地理位置、ASN、开放端口。工具栏图标会替换为网站国家对应的国旗图标。

为保护你的隐私，扩展只发送网址信息。

Privacy Policy



它是如何工作的?

使用这个扩展后，在搜索引擎结果页，方便你查看每个网址都是属于谁的，在具体的网址标签页，显示域名IP信息和国旗图标。

在你浏览网站时，插件会获取该网站访问的IP信息，此时插件会请求域名信息到服务器，获取该域名的所属人信息和IP信息，根据返回的信息展示域名所属人和国旗图标等信息。

在搜索引擎结果页，插件会获取当前结果页所有的网址列表，然后请求该域名列表到服务器，只返回所属人信息，此时不会请求IP信息。返回的结果信息会缓存到本地存储中，下次再请求时会从缓存中加载。在选项中，你可以清除缓存。

自定义规则你可以导入或导出，方便你在不同浏览器之间切换数据。

由于chrome应用商店的严格审核原则，我们的扩展不会收集你的敏感信息，仅限于向服务器请求时发送域名信息.

在搜索引擎结果页显示每项网址对应的所属人信息及位置排名序号

在SERP结果中，插件将会请求服务器获取每项域名的所属人信息，如果该域名已经设置了自定义规则，则不会发送域名信息到服务器。

同时插件会在每个搜索结果中显示当前位置排名序号数字。

弹出框显示域名国旗图标和IP相关信息

浏览网页时，会显示当前网站的IP信息及域名所属人信息。IP信息包含：IP地址、地理位置、ISP信息、ASN信息、端口信息。根据地埋位置信息，会展示相应国家的国旗图标。

自定义规则，保护你的隐私数据不被泄露

如果你不想域名对应的所属人信息保存到服务器，可以添加自定义规则，在本地保存域名对应关系，这些数据不会上传到服务器。

自定义规则优先级最高，已经在自定义规则中的域名，不会再次向服务器请求该域名信息。

200+ 精美的国旗图标

覆盖所有的国家，每个图标都精心制作.



自定义规则格式

在选项中的自定义规则中，你可以定义每个域名对应的所属人信息，这将会在SERP结果中展示出来

自定义规则将会覆盖服务器规则，向服务器请求域名信息时，如果已经有自定义规则则不会再次请求该域名信息。

每行一条规则，域名和名字用空格区分开，格式如下： 

leadscloud.github.io SA
wikipedia.org Wiki
域名会按照最小匹配规则，例如github.io将会匹配leadscloud.github.io。


需要注意的注意事项：

  • 本功能显示的是浏览器里对网站进行访问的 IP，如果您挂了代理，显示的将是代理服务器的 IP；
  • 由于浏览器 API 限制，可能要等到网页完全加载完后，才可以显示相关信息。
  • 服务器端解析您所在浏览的域名对应IP ，这种情况受限于服务器 DNS 设置，仅供参考
